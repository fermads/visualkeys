VisualKeys

VisualKeys is a jQuery plugin for ... 1k you can give your users an alternative way to control your web application without the mouse CMS .. and applications users have to execute repetitive tasks like CMSs or any system administration.

Demo

  1. Hit ctrl+space. A menu will appear at the bottom of this page.
  2. Hit v to execute the "view source code" shortcut and see the code bellow.
  3. That's it! You can configure any action to any key.
    You can also alt+shift+v as a faster shortcut.
  4. Try more shortcuts by pressing ctrl+space again.
+ view code for this demo

Usage

Just add visualkeys style and script to your <head>. jQuery should (probably already) be there too. <link rel="jquery.visualkeys-1.0.css" />
<script src="jquery-1.4.2.min.js"></script>
<script src="jquery.visualkeys-1.0.min.js></script>

You also can...

  • change the appearence of the menu by editing visualkeys.css
  • bind esc to a exec:function(). It'll be executed right away, not showing the menu.
  • the hide:value can be used to turn a shortcut visible/hidden based on a context. What? How?
  • open the menu and mouse click a shortcut. It'll be executed.
  • as an alternative to ctrl+space you can use f9 or shift+space to open the menu.
  • unbind and remove visualkeys with $.visualkeys('remove').
